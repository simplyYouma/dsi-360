# Fichiers statiques (servis à la racine)

Dépose ici les fichiers servis tels quels par Vite, sans import : **`favicon.svg`**,
images de partage, `robots.txt`… Accessibles à l'URL racine (ex. `/favicon.svg`).

Pour les logos **importés dans l'interface**, utilise plutôt `src/assets/brand/` (voir son README).
